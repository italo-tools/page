Responsividade:

Meus sites são responsivos, se ajustando bem na maioria das telas, sem dor de cabeça. Se a tela for maior que 320px, pode ficar tranquilo que tudo vai rodar suave. O único desafio aparece nos portais Android e iPhone que têm aquela tela de 240px, aí a coisa complica um pouco. Mas fora isso, tá tudo certo!

Teste rapido de responsividade?

Limitações:

Meus sites são leves, rápidos e feitos do zero, sem o uso de frameworks. Meu foco é aprimorar cada vez mais meu conhecimento em Front-End, criando interfaces modernas com CSS puro e JavaScript puro, garantindo total controle do código. Ainda estou evoluindo no backend, então meus serviços são voltados para páginas bem estruturadas, responsivas e personalizadas para cada projeto.
